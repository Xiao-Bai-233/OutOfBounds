## 🔧 关键参数速查

### UIPhysicsElement 默认参数
```csharp
mass = 1f;                    // 质量
drag = 1.5f;                  // 阻力（减速速度）
bounciness = 0.2f;            // 弹性（低弹性防弹飞）
friction = 0.3f;              // 摩擦力
useGravity = true;            // 保留但ApplyGravity已禁用
gravityScale = 0.3f;          // 不生效（真空环境）
```

### Layer配置
- UI元素层：`UI_Physics`
- 地面层：`Ground`
- 需要在Physics2D设置中启用UI_Physics与Ground的碰撞

## 🎮 当前可测试功能

1. **基础拖拽**
   - 运行场景
   - 左键按住UI元素拖动
   - 释放后物体会按释放速度运动并减速

2. **固定模式**
   - 右键点击UI元素
   - 物体会变灰并固定不动
   - 再次右键解除固定

3. **碰撞测试**
   - 拖拽UI元素撞向其他物体
   - 观察反弹效果
   - 测试边界碰撞

---

## ⚠️ 已知限制

1. 需要场景中有带Collider2D的地面/墙壁才能测试碰撞
2. 边界约束只在Canvas范围内有效
3. 物体运动速度受drag影响较大（1.5会让物体很快停止）
